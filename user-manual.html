Buddy Planner User Manual

Your complete companion guide to utilizing the ADHD-focused printable planner suite—from managing personalized color themes to digital backup, local device sync, floating pen tools, and physical printing.

Need quick support? Email us at: buddyplannerpk@gmail.com

1. Quick Start Guide

Get started and configure your workspace in under a minute using these simple setup steps:

1

Open the App

Double-click or launch index.html inside any modern web browser (Google Chrome, Microsoft Edge, Mozilla Firefox, or Safari).

2

Pick a Theme

Select from our range of 18 themes (Feminine, Masculine, or Gender-Neutral) to apply colors instantly across all pages.

3

Choose a Planner

Click on any planner sheet card (Daily Focus, Weekly Overview, Monthly Calendar, Habit Tracker, etc.) to start editing.

4

Fill & Print

Fill in details, tick checklists, utilize drawings, and print to physical Letter/A4 paper or export clean PDFs.

💡 Pro-Tip: The planner suite operates completely locally. All choices, text entries, and checkmarks are saved automatically in your browser's local store.

4. All 8 Planner Sheets — Complete Guide

Learn the functions and custom blocks mapped to each of the 8 dedicated planner pages:

Daily Focus

Features an hourly timeline (5:00 AM - 4:00 AM), a Boring Tasks prioritization list, self-care tracker (12 icons), upcoming deadlines sync, brain dump slot, and main habit goals.

Weekly Overview

A 7-day grid tracking priorities for each weekday. Includes a custom weekly focus sidebar and a bottom habit check-in tracker matching key milestones.

Monthly Calendar

A clean 35-day grid. Dynamically pulls country-specific holiday markers. Clicking the Month Name header opens a navigation helper to jump directly to any month/year.

Brain Dump

Leverages the 2x2 Eisenhower Matrix to categorize tasks by urgency and importance (Urgent & Important, Less Urgent, Delegate, Eliminate), alongside a large notes section.

Habit Tracker

Track up to 12 habits over a 31-day timeline. Click cells to cycle state: blank, Done (green), Missed (red), Partial (yellow), and Not Applicable (grey).

Dopamine Menu

Organizes energy-boosting actions across key groups (Quick Wins, Sensory Treats, Social Connection). Great for handling burnout with built-in "Stuck" suggestion helpers.

Expense Tracker

Category-based tracker mapping food, travel, bills, fun, and other expenses. Type your values and let the sheet compute subtotals and grand totals dynamically.

Quick Reminders

Tracks alerts and time stamps. Features an AM/PM time selector with 5-minute incremental controls and clean glassmorphism prompt designs.

5. Using Interactive Fields

Buddy Planner pages are designed to be interactive web components that save your state on the fly:

  • Text Entry Fields: Click inside any write-in section, checklist row, or notes pad to edit content. Click out or stop typing to save instantly.
  • Checkboxes: Tapping checks off tasks, rendering crossed lines and marking items complete.
  • State-cycling Cells: Tapping days in the Habit Tracker cycles colored statuses.
  • Subtotal Computations: Entering numbers inside the Expense Tracker updates values automatically.
⚠️ Data Persistence: Clear browser cache or resetting browser data deletes planner files. We recommend downloading manual JSON files regularly to avoid losing progress.

6. Using the Planner on Mobile Devices

The entire layout is optimized for mobile touchscreens, tablets, and phones:

8. Buddy Suggests — When You Feel Stuck

ADHD paralysis and overwhelm can make initiating tasks difficult. The built-in "I'M STUCK!" feature helps break this cycle:

  1. Locate the Buddy Suggests card on the primary Dashboard or tap the "Stuck" button on the Dopamine Menu.
  2. Clicking generates a single, simple activity suggestion (e.g. "Drink a glass of water", "Open a window for 1 minute").
  3. Not feeling the prompt? Click "Another Idea" to swap suggestions.
  4. Ready to start? Select "I'll Do This!" for encouragement.

9. Backups & Data Management

Manage your saved database profiles locally. The backup utility includes these commands:

Control Button Purpose & Action
Backup Now Saves a duplicate copy of all planner parameters to local browser memory.
Restore Latest Restores your planner data from the most recent backup snapshot.
Export JSON Downloads a local file backup (`.json`) containing all data. Save this on your computer or cloud storage.
Import JSON Uploads a previously saved JSON file to restore your planner data on any device.
Clear All Clears all browser memory, local storage records, and theme options. Cannot be undone.

10. Installing as an App (PWA)

Run Buddy Planner as a native offline application on your desktop or mobile home screen:

Desktop Install (Chrome / Edge / Safari)

  1. Open the dashboard homepage.
  2. Click the Install monitor icon in your address bar.
  3. Confirm the app prompt to add to your desktop applications menu.

Mobile Install (iOS / Android)

  1. On iOS: Open in Safari, tap the Share button, and choose "Add to Home Screen".
  2. On Android: Open in Chrome, tap the menu (three dots), and select "Install App".

11. Printing & PDF Export

Physical prints are a great way to stay organized. Check these settings for high-quality prints:

Print Settings Check-list

  • Click the Print Page action button.
  • Set Destination to Save as PDF or select your printer.
  • Orientation: Portrait.
  • Paper size: Letter or A4.
  • Margins: Set to None or Default.
⚠️ Crucial Print Option: You MUST enable "Background graphics" under the print dialog's advanced settings to print colors, graphics, and backgrounds.

15. Frequently Asked Questions

Will my selected theme stay active when I come back?

Yes. Your theme selection is saved in browser storage and will load automatically on your next visit.

Can I use the planner without printing?

Absolutely. You can fill out sheets digitally, type tasks, check off items, and use drawing tools directly in your browser. All data saves automatically.

Why do theme colors not appear in my print preview?

Most browsers hide background graphics to save ink. Make sure to check the "Background graphics" option in your print settings.

What happens if I clear my browser data?

Clearing browser cache or cookies will erase local storage, resetting all planner pages. Use the Export JSON option to keep a backup file on your computer.

How do I transfer data to another device?

Export your data as a `.json` backup file from the dashboard, transfer it to the target device, and choose Import JSON on the new device.
