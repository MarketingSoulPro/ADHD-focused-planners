Quick Start

The Buddy Planner dashboard lets you select a visual theme and open any printable planner sheet. On mobile, the interface includes theme tabs, planner navigation icons, and a floating tool button.

1

Choose a theme

Select a background style from the dashboard or from the in-planner theme controls.

2

Open a sheet

Pick Daily, Weekly, Monthly, Brain Dump, Habit Tracker, Expense Tracker, or another sheet.

3

Edit or mark items

Tap fields, checkboxes, icons, and planner areas to write, track, and organize your day.

4

Print or save PDF

Use the print button and enable background graphics for the best visual result.

Dashboard Guide

The dashboard is the starting point for the planner suite. It is designed to help you quickly choose a visual style and open the exact planner sheet you need.

  • Select Background Theme lets you pick one of the available design themes.
  • Choose Planner Sheet opens the planner page you want to use.
  • Current Theme shows the active theme name and color swatch.
  • Printable badges indicate planner pages intended for printing or PDF export.

Using the Planner on Mobile

On mobile, the planner interface is optimized with tabs and compact controls.

  1. Tap Tools to open or close the floating tool button/toolbar.
  2. Tap Select Themes to open or close the mobile theme selector.
  3. Use the planner navigation icons to move between pages such as Daily, Weekly, Brain Dump, and Expense Tracker.
  4. Tap the month tabs to switch between months and update planner content.

Floating Tools

The floating tool button gives quick access to drawing and annotation tools. It starts minimized and can be expanded when needed.

  • Pen for handwriting or marks
  • Eraser for removing strokes
  • Stickers/emojis for visual markers
  • Color and size controls
  • Undo and clear actions

Printing Tips

For the best output, use your browser print dialog and make sure Background graphics is enabled.

  • Destination: save as PDF or select your printer
  • Layout: portrait
  • Paper size: Letter or A4
  • Margins: none or default, depending on your printer
  • Check preview before printing multiple pages

Frequently Asked Questions

Will my selected theme stay active?

Yes. The selected theme is stored locally in your browser and should remain active when you reopen the app.

Can I use the planner without printing?

Yes. You can interact with planner pages digitally, then print or export when ready.

Why do theme colors not appear in print preview?

Enable Background graphics in your browser print settings before printing or saving to PDF.
