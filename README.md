# ADHD-Focused Planners

An interactive web-based planner application designed specifically for individuals with ADHD. Features multiple planning pages and beautiful theme options to help with organization, time management, and task tracking.

## Features

- 🎨 **12 Beautiful Themes** - Choose from various aesthetic themes including abstract brushstrokes, botanical garden, dark slate, geometric zen, and more
- 📅 **Daily Focus** - Set and track your daily priorities
- 🧠 **Brain Dump** - Quick capture of thoughts and ideas
- 📊 **Habit Tracker** - Monitor habits and build streaks
- 💰 **Expense Tracker** - Keep track of spending
- 🎯 **Quick Reminders** - Fast note-taking
- 📈 **Weekly Overview** - Plan your week at a glance
- 🗓️ **Monthly Calendar** - View and plan your month
- 🎪 **Dopamine Menu** - Activities to boost mood and energy

## Pages

- `daily-focus.html` - Daily planning and priority setting
- `brain-dump.html` - Quick thought capture
- `habit-tracker.html` - Habit tracking and monitoring
- `expense-tracker.html` - Financial tracking
- `quick-reminder.html` - Quick notes
- `weekly-overview.html` - Weekly planning
- `monthly-calendar.html` - Monthly view
- `dopamine-menu.html` - Mood-boosting activities

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Or run the server: `node server.js` (then open `http://localhost:3000`)
4. Choose your favorite theme and start planning!
